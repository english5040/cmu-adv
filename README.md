CMU-MUD
=======

A "Multi-User Dungeon" (if you think CMU is a dungeon...) Text Adventure!

[![mit-license](http://img.shields.io/:license-MIT-blue.svg?style=plastic)](http://bescott.mit-license.org)

---
This is a repository for the soon-to-be coolest text adventure on campus! *CMU-MUD* is written in TADS3, and compiles to a *.t3 format, readable by the [frob](https://github.com/realnc/frobtads "FrobTADS") interpreter.
